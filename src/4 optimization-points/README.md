## 核心优化思路

1. 多机并行：能够充分利用多机的资源
2. 单机多核并行：能够充分利用单机上多核的资源
3. 充分利用单核的性能：
    - 3.1 减少 指令数：选择效率更好的执行策略，数据结果，算法 等
    - 3.2 减少 每个指令需要的周期：SIMD, CPU Cache，分支预测 等
